# Wildcard Search

You can use the asterisk (`*`) as a wildcard to represent any number of characters in a filename. For example, `grep "search_term" *.txt` will search for "search_term" in all files ending in `.txt`.

There are 50 files in this directory. One of them contains the flag. Instead of checking each one individually, use a wildcard to search them all at once!

## Allowed commands:

* grep
* ls
