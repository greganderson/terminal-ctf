# Pipes & Counting

Pipes (`|`) allow you to take the output of one command and use it as the input for another. This is one of the most powerful features of the terminal.

In this directory, there is a file called `data.txt`. Your goal is to count how many times the word **"banana"** appears in the file. 

The flag is the **number** of occurrences.

## Hint:
Use `grep` to find the lines, and pipe the result into `wc -l` to count them.

## Allowed commands:

* grep
* wc
