# Permissions

In Linux, every file has permissions that determine who can read, write, or execute it. You can see these permissions using `ls -l`.

There is a file called `secret_flag.txt`, but it's currently locked (no permissions). You need to change the permissions using the `chmod` command to make it readable.

## Hint:
Try `chmod +r secret_flag.txt` to add read permissions.

## Allowed commands:

* ls
* chmod
* cat
